  • Entry = Status — re-entering when connected shows "Connected · IP", not a list. The not-connected action is Connect (scanning is internal).
  • One row per network — mesh / duplicate APs merged by SSID, strongest signal wins.
  • Empty list → "No networks found" plus Rescan (never a dead end); Other network… handles hidden / manual SSIDs.
  • Password: SSID shown, show-password toggle, keyboard-OK = Connect, Cancel → list. Connecting times out → failure + Try again.
  • No WPS (deprecated; Ecobee/Nest don't use it) — the advanced path is manual SSID entry. Out of scope: captive portal, enterprise/802.1X.
