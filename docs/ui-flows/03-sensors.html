Look again
  • Plain words only. "Rooms / sensors", "someone here", "following", "use this room", "nudge the reading" — never broker / roster / topic / MQTT / HA.
  • Lead with what it's doing: "Now using 21.4° · 2 rooms · following the living room" — the temperature it's acting on and why.
  • Each room, one clear choice: a switch to use it or not. Off rooms dim; a room that stops responding is flagged "check it" and dropped automatically.
  • This thermostat (its own sensor) can't be switched off — the always-there fallback.
  • Room detail adds "follow when occupied" and a temperature nudge (calibration), in words, not numbers-jargon.
  • Connecting to the home system is silent + automatic; the technical address hides under Installer → Advanced. Choices are saved automatically and shared with the rest of the system.
