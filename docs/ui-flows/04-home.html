  • Mode-aware target: Heat → one Heat target · Cool → one Cool target · Auto → both (a range) · Off → none. You never see a heat number while cooling.
  • Logo — the graphic mark sits before the "SlyTherm" wordmark on the left.
  • Presence rule (the "reading from" line): Present when someone's in the room → Last entered X min ago (minutes, under 1 h) → Last entered Y hr ago (rounded hours, 1–3 h) → after 3 h with no presence anywhere → Nobody home · averaging N rooms.
  • Legible setpoint: single big value with 54px +/- — fixes the overprinted "8.0°".
  • Diagnostics moved off the top: the status row is just an online dot + outdoor temp; WiFi/MQTT/bus detail lives on the Diag → Links section.
