Dettson ClimateTalk Thermostat

Web firmware installer — ESP32-DevKitC bench images

Flash a pre-built firmware image straight from this page over USB. Plug the ESP32-DevKitC into this computer, pick an image, and click Install. No toolchain required.

1 · Choose a firmware image

Safety note. The thermostat image boots DEMANDS DISABLED: the actuator only logs, CT-485 TX is compiled out, and no relay pin is configured. Flashing convenience never bypasses the Phase 2/3 safety gates — live bus or relay output requires a deliberate custom build with each gate opened (see docs/04-safety.md).

2 · Connect and install

Requires a browser with Web Serial: Chrome or Edge on desktop, over https:// (or localhost). Firefox and Safari are not supported. If the device is not detected, install the board's USB-serial driver (CP210x or CH340, depending on the DevKitC variant).

3 · First boot

What to expect on the serial console (115200 baud — use the Logs & Console option in the install dialog, or pio device monitor). Both images need Wi-Fi credentials compiled in to do anything network-facing; a web-flashed image without them falls back accordingly. The thermostat image with no sensor input keeps its boot gate closed (boot-to-no-demand) and raises the boot-grace alarm after 120 s — that is the system working as specified, not a bug. See the first-boot walkthroughs in src/main_thermostat.cpp (bench quickstart) and src/main_sniffer.cpp.
